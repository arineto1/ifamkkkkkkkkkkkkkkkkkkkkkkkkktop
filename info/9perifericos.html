Hardware: Periféricos

1. Definição e Arquitetura de E/S

Periféricos são dispositivos conectados ao Barramento de E/S que permitem a interação entre o processamento interno e o mundo externo. Essencial entender a divisão:

  • Dispositivo de E/S (Hardware): O componente físico (teclado, monitor).
  • Controlador de Dispositivo (Chipset/Placa): Responsável por traduzir os sinais elétricos do barramento em comandos para o dispositivo.
  • Device Driver (Software): Código que permite ao Sistema Operacional conversar com o controlador.

2. Técnicas de Comunicação (Ponto de Prova!)

Como a CPU gerencia a enorme diferença de velocidade entre ela (GHz) e um HD ou Teclado (ms)?

Técnica Mecanismo Desvantagem / Vantagem
E/S Programada A CPU fica em loop perguntando se o dado está pronto (Polling). Desperdiça tempo da CPU (Busy Waiting).
E/S por Interrupção O dispositivo envia um sinal (IRQ) quando termina a tarefa. Libera a CPU para outras tarefas enquanto o periférico trabalha.
DMA (Acesso Direto) O Controlador DMA assume o barramento e move dados entre Periférico e RAM. A CPU só é interrompida no final do bloco. Ideal para grandes volumes (Discos).

3. Mapeamento e Endereçamento

A CEFETMINAS costuma cobrar como a CPU identifica um periférico:

  • Memory-Mapped I/O: O periférico ocupa um endereço na memória RAM. Usa as mesmas instruções (LOAD/STORE).
  • Isolated I/O: Existe um espaço de endereçamento separado. Exige instruções de CPU exclusivas (IN/OUT).

4. Interfaces Externas: O Padrão USB

5. Armazenamento e Conectores

  • M.2: Conector físico versátil (pode ser SATA ou PCIe).
  • NVMe: Protocolo otimizado para memória Flash, rodando sobre PCI Express.
  • Hot Swapping: Capacidade de remover/inserir o periférico com o sistema ligado (Ex: USB, SATA).

6. Periféricos de Saída (Vídeo e Impressão)

Processo de Impressão a Laser (As 6 Fases)

  1. Limpeza: Remove resíduos de toner do cilindro.
  2. Condicionamento (Carga): Aplica carga negativa no cilindro.
  3. Escrita (Exposição): O Laser descarrega as áreas onde haverá imagem.
  4. Revelação: O toner (negativo) gruda nas áreas descarregadas (positivas).
  5. Transferência: O toner passa para o papel.
  6. Fusão: O Fusor aquece e prensa o toner no papel definitivamente.

Vídeo

  • HDMI vs DisplayPort: O DP suporta Daisy Chaining (conectar vários monitores em um único cabo via MST).
  • Refresh Rate: Medido em Hz. Tecnologias G-Sync/FreeSync sincronizam Hz da GPU com o monitor para evitar "tearing".

⚠️ Radar CEFETMINAS - Não erre isso!

  • Interrupção vs Polling: A banca adora dizer que Polling é mais eficiente. Mentira. Interrupção é mais eficiente para a CPU.
  • DMA e Barramento: O DMA "rouba" ciclos de barramento da CPU (técnica de Cycle Stealing).
  • Híbridos: Impressoras Multifuncionais e Monitores Touch são periféricos de Entrada e Saída simultaneamente.
  • USB-C: Não é uma velocidade, é apenas o formato físico do conector (reversível).
