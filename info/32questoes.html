Questão 01 Aritmética e Precedência

Analise o algoritmo abaixo, desenvolvido na sintaxe padrão do Portugol (VisualG). O objetivo do código é realizar operações mistas entre números inteiros e reais, testando o comportamento dos operadores div, mod e /.

Algoritmo "Calculo_Misto_Precedencia"
Var
   A, B, Aux : Inteiro
   X, Y : Real
Inicio
   A <- 7
   B <- 3
   Aux <- (A + B) * 2
   A <- Aux div B
   X <- (Aux / B) + 0.5
   Y <- (Aux mod B) + X
   Se (Y > 10.0) Entao
      B <- B + 2
   FimSe
   A <- A + B
   Escreva("A=", A, " Y=", Y)
FimAlgoritmo
Questão 02 Condicionais Aninhadas

O trecho de código a seguir avalia a concessão de um benefício com base em idade e tempo de serviço.

Algoritmo "Analise_Beneficio"
Var
   Idade, TempoServico : Inteiro
   Direito, Especial : Logico
   Bonus : Inteiro
Inicio
   Idade <- 55
   TempoServico <- 28
   Bonus <- 0
   Direito <- Falso
   Especial <- Falso

   Se ((Idade >= 60) OU (TempoServico >= 30)) Entao
      Direito <- Verdadeiro
   Senao
      Se ((Idade >= 50) E (TempoServico >= 25)) Entao
         Especial <- Verdadeiro
      FimSe
   FimSe

   Se (Direito) Entao
      Bonus <- 1000
   Senao
      Se (Especial E (NAO Direito)) Entao
         Bonus <- 500
         Se (TempoServico > 27) Entao
             Bonus <- Bonus + 200
         FimSe
      Senao
         Bonus <- 100
      FimSe
   FimSe
   Escreva(Bonus)
FimAlgoritmo

Questão 06 Vetores e Índices 
Algoritmo "Vetor_Acumulativo"
Var
   V : Vetor[1..6] de Inteiro
   K : Inteiro
Inicio
   Para K de 1 ate 6 Faca
      Se (K mod 2 = 0) Entao
         V[K] <- K
      Senao
         V[K] <- 0
      FimSe
   FimPara
   Para K de 2 ate 5 Faca
      V[K] <- V[K-1] + V[K] + V[K+1]
   FimPara
   Escreva(V[4])
FimAlgoritmo
Questão 07 Matrizes e Diagonais

   M : Vetor[1..3, 1..3] de Inteiro

   Res <- 0
   Para L de 1 ate 3 Faca
