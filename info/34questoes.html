Questão 01: Tipagem e Inferência em Python

Considere o comportamento da linguagem Python (versão 3.x) em relação à tipagem dinâmica e à manipulação de variáveis. Analise o seguinte trecho de código:

var_a = 10 var_b = "5" var_c = 2.5 # Expressão 1 resultado_1 = var_a * 2 + int(var_b) # Expressão 2 resultado_2 = (var_a > 5) and (var_c < 3.0) or (var_b == 5) # Expressão 3 var_d = [1, 2, 3] var_e = var_d var_e.append(4)

Com base na execução desse script, assinale a alternativa CORRETA:

Resolução

Em Python, listas são objetos mutáveis e a atribuição var_e = var_d cria uma nova referência para o mesmo objeto na memória. Portanto, var_e.append(4) altera a lista que ambas as variáveis apontam. resultado_1 é 25 pois (10 * 2) + 5 = 25.
Questão 02: Precedência de Operadores em C

Analise o programa a seguir na linguagem C:

int main() { int x = 5; int y = 2; int z; z = (x++ * 2) + (--y % 2); printf("X=%d, Y=%d, Z=%d", x, y, z); return 0; }

Assinale a alternativa que apresenta a saída exata produzida pela execução deste código:

Resolução

1. x++ usa 5 e depois x vira 6. (5 * 2 = 10).
2. --y reduz y para 1 e usa 1. (1 % 2 = 1).
3. z = 10 + 1 = 11.
Resultado final: X=6, Y=1, Z=11.
Questão 03: Estruturas Condicionais em Shell Script
#!/bin/bash VALOR=15 LIMITE=10 STATUS="INATIVO" if [ $VALOR -gt $LIMITE ] && [ $STATUS != "ATIVO" ]; then VALOR=$((VALOR + 5)) fi echo "Valor Final: $VALOR"

Qual será o resultado impresso no terminal?

Resolução

A condição $VALOR -gt $LIMITE (15 > 10) é verdadeira. A condição $STATUS != "ATIVO" ("INATIVO" != "ATIVO") também é verdadeira. O bloco incrementa VALOR para 20.
Questão 04: Avaliação de Expressões Lógicas (Visualg)

Valores: A <- 3, B <- 7, C <- 4.

Expressão: ((A + C) > B) OU ((B MOD 2) = 1) E NAO (A > C)

Resolução

1. (3+4) > 7 -> 7 > 7 é Falso.
2. (7 MOD 2) = 1 -> 1 = 1 é Verdadeiro.
3. 3 > 4 é Falso -> NAO(Falso) é Verdadeiro.
4. Falso OU (Verdadeiro E Verdadeiro) -> Falso OU Verdadeiro = Verdadeiro.

Questão 06: Lógica Proposicional

"Se o usuário é administrador E a senha é forte, então o acesso é liberado." Qual a negação lógica?

Resolução

A negação de P -> Q é P e não Q. Mantém-se o antecedente ("Usuário admin e senha forte") e nega-se o consequente ("Acesso NÃO liberado").

Questão 08: Avaliação de Curto-Circuito
int a = 0; int b = 10; int c = 0; if (a != 0 && (b = b + 5) > 10) { c = 1; } else { c = 2; } printf("%d %d", b, c);

Resolução

Como a != 0 é falso, o operador && interrompe a avaliação (curto-circuito). A atribuição b = b + 5 nunca é executada. b continua 10. O bloco else define c=2.
Questão 09: Recursividade
Função Calc(n) Se n <= 1 Então Retorne 1 Senão Retorne n + Calc(n - 2) FimFunção

Qual o valor de Calc(6)?

Resolução

Calc(6) = 6 + Calc(4)
Calc(4) = 4 + Calc(2)
Calc(2) = 2 + Calc(0)
Calc(0) = 1
Resultado: 6 + 4 + 2 + 1 = 13.
