1. Definição e Características Essenciais

A Lógica de Programação é a aplicação da Lógica Matemática na construção de algoritmos. Para a FCM, entenda que um programa é uma sequência finita de instruções que manipulam dados através de expressões.

Algoritmo: Conjunto de passos não ambíguos e finitos. Para concursos de Informática, a atenção deve estar na Eficiência (complexidade) e na Determinidade (mesma entrada sempre gera mesma saída).

2. Variáveis: O Coração dos Dados

Espaços nomeados na memória RAM. Características cobradas:

  • Identificador: Deve começar com letra ou underscore (_). Em Java/C, Case-Sensitive (Idade != idade).
  • Tipagem Estática: O tipo é definido na compilação (Java, C, C++).
  • Tipagem Dinâmica: O tipo é definido em tempo de execução (Python, JS).

Peguinha da Banca: L-Value vs R-Value

A FCM costuma cobrar se uma expressão pode receber atribuição.
- L-Value: Endereço fixo (variável). Ex: x = 10; (Correto).
- R-Value: Valor temporário. Ex: (x + 5) = 20; (ERRO! Não se atribui a um resultado de soma).

Ciclo de Vida e Escopo

Escopo Visibilidade Tempo de Vida
Local Apenas dentro da função/bloco. Criada na chamada, destruída no retorno.
Global Todo o programa/arquivo. Enquanto o programa rodar.
Estática (static) Local à função. Persiste entre chamadas da função.

3. Operadores e a Tabela de Precedência

O "divisor de águas" em provas de nível superior. A FCM exige precisão na ordem de execução.

Operadores Bitwise (Pouco estudados, muito cobrados)

Esses operadores trabalham bit a bit. Frequentemente aparecem em questões de "Qual o resultado da execução?".

& (AND): 1 se ambos forem 1.
| (OR): 1 se um for 1.
^ (XOR): 1 se forem diferentes.
~ (NOT): Inverte os bits.
<< (Left Shift): Multiplica por 2.
>> (Right Shift): Divide por 2.

4. Análise de Tendências e "Peguinhas" CEFETMINAS

A. Curto-Circuito (Short-Circuit Evaluation)

Em Java e C, se a primeira parte de um && for FALSA, a segunda parte sequer é testada.
Exemplo de Prova: if (a == 0 && (b/a > 1)). Se a for 0, o programa NÃO dá erro de divisão por zero, pois o segundo termo é ignorado.

B. O Ponto e Vírgula Fantasma

Observe atentamente o final das linhas de controle:

for (int i=0; i < 10; i++);
{ printf("%d", i); }

O ; finaliza o loop. O bloco abaixo será executado apenas UMA VEZ com o valor final de i.

C. Promoção de Tipos e Divisão

Para a banca, float x = 5 / 2; resulta em 2.0 e não 2.5.
Isso ocorre porque a operação entre dois inteiros ocorre primeiro (resultado 2) e depois é promovida para float.

5. Diferenças Cruciais EBTT: C vs Java

Recurso Linguagem C Linguagem Java
boolean Não existe tipo nativo puro (usa-se int). Tipo boolean obrigatório em if/while.
Iniciação Variáveis locais contêm lixo de memória. Variáveis locais EXIGEM inicialização.
Ponteiros Acesso direto à memória (&, *). Referências ocultas (Manipulação indireta).
Switch Aceita apenas integrais (int, char). Aceita int, char, String e Enums.
