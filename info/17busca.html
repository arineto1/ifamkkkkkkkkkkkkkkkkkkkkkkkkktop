1. Definição e Conceitos Essenciais

Os mecanismos de busca são sistemas de software projetados para realizar buscas de informações na World Wide Web. Para a prova do IFAM, entenda que a busca não ocorre na internet "ao vivo", mas sim em um Índice previamente construído.

Componente Função Técnica Termo de Prova
Coletor Visita páginas seguindo links. Crawler, Spider ou Bot.
Indexador Organiza as palavras em um banco de dados. Índice Invertido.
Interface Onde o usuário digita a Query. Search Engine Query.

2. Divisões da Web e Acessibilidade

A CEFETMINAS adora cobrar a distinção entre o que é visível e o que é oculto para os buscadores tradicionais.

  • Surface Web: Conteúdo indexável (páginas HTML públicas).
  • Deep Web: Conteúdo não indexado por buscadores comuns. Inclui bancos de dados, extratos bancários, e-mails e páginas protegidas por senhas. Não é necessariamente ilegal.
  • Dark Web: Uma pequena parte da Deep Web, acessível apenas via softwares específicos (como o Tor), focada em anonimato extremo.

4. Outros Buscadores e Meta-buscadores

Não estude apenas o Google. A banca pode citar alternativas:

  • Bing (Microsoft): Possui integração com IA (Copilot) e operadores como ip: e language:.
  • DuckDuckGo: Foco total em privacidade (não rastreia o usuário).
  • Meta-buscadores: Não possuem banco de dados próprio. Eles enviam a consulta para vários buscadores ao mesmo tempo e agregam os resultados. Exemplo clássico: Dogpile.

5. Tendências de Cobrança (CEFETMINAS)

Padrão da Banca para cargos EBTT:

  1. Sintaxe Errada: Eles colocam site : ifam.edu.br com espaço antes dos dois pontos para te confundir.
  2. Case Sensitivity: Operadores como OR devem ser maiúsculos. A busca comum (palavras) NÃO diferencia maiúsculas de minúsculas.
  3. Stop Words: Palavras vazias (e, de, o, a) são ignoradas pelo buscador, a menos que estejam entre aspas.
  4. Lógica Booleana: Cobram a compreensão de que o espaço entre palavras no Google funciona como um AND implícito.

6. Flashcard de Revisão Rápida

O que é Indexação?

Processo de organizar o conteúdo coletado pelo crawler para permitir buscas ultra rápidas.

Diferença Deep vs Dark?

Deep é o não indexado (legítimo ou não). Dark é o oculto intencional que exige redes como TOR.

Operador Coringa (*)

Substitui uma palavra inteira. Útil para quando você esqueceu parte de uma frase.

Yahoo Search

Atualmente utiliza a tecnologia de busca (indexação/ranking) do Bing.
